## Project Overview
This project analyzes NSC EESP consumer data using a database-driven dashboard and machine learning to predict future demand, load patterns, phase requirements, and capacity planning.

## Tech Stack
- Python
- Streamlit
- SQLite
- Machine Learning (Random Forest)
- GitHub

## Features
- Area-wise request analysis
- Consumer category insights
- Phase-wise demand analysis
- Future load prediction using ML
